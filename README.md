# E-CommerceApp-Ui-Redising

* Giriş Sayfası
* Kayıt Sayfası
* Anasayfa (Üç ayrı Flat List - Search Bar - Categories - Products)
* Ürün Önizleme
* Satın Alma 
* Tüm Fonksiyonları Çalışır Hale Getirme
